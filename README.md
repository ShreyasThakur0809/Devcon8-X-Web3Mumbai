# Devcon8 × Web3Mumbai 🇮🇳

> A grassroots Road to Devcon 8 initiative — bringing Ethereum to Mumbai's builders, students, and founders ahead of Devcon 8.

---

## About

**Web3 Mumbai** is a community-led initiative running hackathons, workshops, and meetups across Mumbai's engineering colleges and builder communities. This repository documents our **Road to Devcon 8** program — events, resources, and on-chain outcomes built in the lead-up to Devcon 8 (November 2026).

We are supported by the **Ethereum Foundation's Road to Devcon Ecosystem Program**.

---

## Mission

Convert Mumbai's traditional technical talent into active Ethereum contributors — one campus at a time.

## On-Chain Outcomes

Each event is designed to produce verifiable, on-chain proof of participation:

-  Sepolia testnet wallet funding for new devs
-  POAP mints for event attendees
-  Open-source GitHub repositories from hackathon participants
